# Rasmlar papkasi

Maktabning haqiqiy rasmlarini shu papkaga joylang, so'ng `lib/data.ts` faylidagi
`image` qiymatlarini `/images/fayl-nomi.jpg` ko'rinishida yangilang.

Tavsiya etilgan rasmlar:

| Fayl nomi (taklif)      | Joyi                          | O'lcham (taxminiy) |
| ----------------------- | ----------------------------- | ------------------ |
| `maktab-binosi.jpg`     | Bosh sahifa — Hero            | 1200×900           |
| `direktor.jpg`          | Bosh sahifa — Direktor murojaati / Rahbariyat | 600×800 |
| `raimqulov.jpg`         | Maktab haqida — R. Rayimqulov | 600×800            |
| `oqituvchi-*.jpg`       | O'qituvchilar kartochkalari   | 600×750            |
| `yangilik-*.jpg`        | Yangiliklar                   | 1200×750           |
| `galereya-*.jpg`        | Galereya                      | 1000×1000          |

> Rasmlarni siqilgan (optimallashtirilgan) holatda joylang — sayt tezligi uchun muhim.
> Next.js `Image` komponenti qolgan optimizatsiyani avtomatik bajaradi.
